# Paint: draw a Symphonee-ish mark

app: mspaint
difficulty: tier1
setup: Open Microsoft Paint with a fresh blank canvas sized around
800x600. Default brush (pencil, black) is fine.

goal:
Draw a simple orchestra-conductor-style emblem for an app called
Symphonee: a short vertical stroke in the middle (the baton), three
small curved strokes fanning out to the right, and the word
"Symphonee" typed below using the text tool. Keep it schematic,
not artistic. Finish when you can see the drawing is complete.

watch-for:
The agent has to use mouse_move + drag to paint, then switch to the
text tool by clicking its icon in the toolbar, then click on the
canvas and type. Expect at least one fumble when the text tool is
picked; the screenshot will show the tool state changed.
