# Word: formatted project note

app: winword
difficulty: tier2
setup: Open Microsoft Word. A blank document. Do not have anything
else selected or highlighted.

goal:
Type a short project status update titled "Weekly Check-In" as a
heading, followed by three bullet points: what shipped, what is
blocked, and next steps. Use Word's heading style for the title
(Home -> Styles -> Heading 1) and bullet list for the three lines.
Save nothing; just get the formatting on screen.

watch-for:
The Home tab on the ribbon is the frequent source of failure. The
agent has to find the Heading 1 thumbnail in the Styles gallery,
then the bullet-list button. Expect multiple screenshots before a
correct click. The visual outcome is a well-formatted page, and the
agent should mention that the heading is now bold/blue and the
bullet marks are visible.
